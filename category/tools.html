Use Google Analytics With Pelican

Posted on 2020-04-13 星期一 in Tools • Tagged with Pelican

基于 Pelican 生成的静态页面，可以使用 Google Analytics 来统计访问数据。

以下是操作步骤，记录在这里备忘：

  1. 登录到 Google Analytics，创建账号；
  2. 添加需要统计的 URL；
  3. 获取 Tracking ID；
  4. 把 Tracking ID 添加到 publishconf.py 里： 

      GOOGLE_ANALYTICS = 'UA-XXXXXX-Y'

  5. 重新编译并发布即可。

参考：Google Analytics With Pelican

Pelican 编译问题

Posted on 2017-12-22 星期五 in Tools • Tagged with Pelican, Python

今天遇到了一个问题：重新编译 pelican 项目时，发现编译生成的 articles 数量要比正常的少很多，而且没有任何出错信息。

经过研究，找到了原因。前段时间更新了 Python 2.7 到最新版本，并且安装了第三方的库，当然包括了 pelican。但 markdown 并没有安装，这导致在编译时 markdown 文件时无法解析。

解决方案很简单，安装 markdown，然后重新编译即可。

pip install markdown

之后又重新看了一遍 pelican 安装说明，明确了 markdown 只属于 optional packages，而不是必选的依赖项。

Find the file by inode number

Posted on 2016-01-22 星期五 in Tools • Tagged with shell

在命令行下会遇到一些文件名里包含有中文、符号或者某些奇怪字符的文件，如果命令的自动补全不完整，键盘操作起来还是有些麻烦的。这时候，可以考虑利用 inode 来定位这样的文件：

1. 使用 ls 显示其对应的 inode 号： 

  ls -i

2. 使用 find 显示其文件名： 

  find . -inum <inode-number>

好吧，拿到文件名了。

iPhone Photo Library "Ghost" Space

Posted on 2015-12-25 星期五 in Tools • Tagged with iOS

最近 iPhone 手机遇到个问题：照片拍了很多了，想清理一下，结果发现即使是删掉了绝大部分照片，照片图库仍然占用了大量的系统存储空间。尝试了好几次，都无法解决。

后来发现，这其实是 iPhone 的一个 Bug。网上已经有了一个临时的解决办法，请参考这个官方论坛里的帖子：https://discussions.apple.com/thread/6560594?tstart=0。

以下是解决步骤：

  • 把手机系统时间改回很久以前；
  • 切换到照片图库，在 “ 最近删除 ” 里面，会看到很多照片；
  • 删除这些照片；
  • 切换到系统设置里，查看系统存储空间变化；
  • 恢复系统时间；
  • OK!

这个 Bug 很奇怪，这个解决办法更奇怪。
