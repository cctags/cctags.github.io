生成 cuda 的 tags 文件

2018-03-07 Wednesday · By cctags · Posted in Tools · 0 Comments

问题：

项目开发过程中，遇到这样的问题：使用 ctags universal 生成 cuda 代码文件的 tags 时，总是有一些函数及其他符号无法生成，影响开发效率。

临时办法：

实在找不到其他办法，临时试了一下按 C++ 文件来处理。

使用到了 ctags 的一个选项 --langmap，指定某些文件按某种语言来处理，这样：

git ls-files | ctags-universal --langmap=c++:+.cu,c++:+.cuh -R -L -

实际试下来，很多原先无法生成的函数及符号，现在都有了。效果还是不错的。
