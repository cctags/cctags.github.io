Remove Trailing Whitespace in Visual Studio

2016-02-24 Wednesday · By cctags · Posted in Tools · 0 Comments

编程习惯里通常会建议删除行尾的空格。在 Visual Studio 里，可以通过 Ctrl+K, Ctrl+\ 的按键操作来实现。

不过每次通过手动的方式，未免有些麻烦，最好还是在保存文件时自动完成。网上找了一下，有个讨论贴子里推荐了 Trailing Whitespace Visualizer，这是个 Visual Studio 的免费插件，可以实现这个功能。

这里是插件的地址：https://visualstudiogallery.msdn.microsoft.com/a204e29b-1778-4dae-affd-209bea658a59。下载后点击安装并重启 Visual Studio 即可。
