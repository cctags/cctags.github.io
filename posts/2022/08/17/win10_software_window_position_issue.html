解决 Windows 10 睡眠唤醒后软件窗口大小和位置有变化的问题

2022-08-17 Wednesday · By cctags · Posted in Tools · 0 Comments

问题

Windows 10 经常碰到这样的情况，从睡眠状态唤醒以后，部分运行中的软件，窗口会被缩小，并且窗口也不在原来的位置，而是会偏向桌面的左上角。

解决

从搜索到的结果来看，是 Windows 注册表里几个参数的设置与实际分辨率不一致引起的问题。以下是调整步骤：

  1. 打开注册表，cmd 里输入 regedit；
  2. 定位到 HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\GraphicsDrivers\Configuration；
  3. 查看 Configuration 下各个项目（及子项目）里的配置，主要是
    • ActiveSize.cx、ActiveSize.cy
    • DwmClipBox....
    • PrimSurfSize.cx、PrimSurfSize.cy
  4. 相应的项，按实际显示器的分辨率来配置。比如，分辨率是 2560x1440，那么宽度相关的设置成 2560，高度相关的设置在 1440；
  5. 重启 Windows 后生效。
