终端与远程之间的文件传递

2011-01-11 Tuesday · By cctags · Posted in Tools · 0 Comments

经常需要在终端和远程服务器之间进行文件传递，以下是几种常用的方法：

  • ssh（文件比较大时，会比较慢）
    • rz/sz https://linux.die.net/man/1/sz
    • rsync https://linux.die.net/man/1/rsync
    • pscp https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html
    • WinSCP
    • Xshell，可以直接拖拽，连 rz/sz 都省了
  • sftp
  • ftp，速度快；需要考虑文件的访问权限；
  • python 小工具：
    • 下载文件，python -m SimpleHTTPServer <port>，以及 wget http://<ip>:port/xxx
    • 上传文件，可以使用 SimpleHTTPServerWithUpload.py
      • https://gist.github.com/UniIsland/3346170
      • https://linuxtoy.org/archives/simple-http-server-with-upload.html
