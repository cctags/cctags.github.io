chrome 导入 portable firefox 收藏夹

2011-09-05 Monday · By cctags · Posted in Tools · 0 Comments

之前用的是 portable firefox，现在慢慢地转到 chrome 了。 在导入收藏夹时遇到了问题。chrome 不知道 portable firefox 收藏夹的位置，折腾了好一会儿，找到个办法：

  1. 将 firefox 的收藏夹导出为 bookmarks.html；
  2. 将 bookmarks.html 导入 IE；
  3. 在 chrome 里，导入 IE Favorites；
  4. OK!
