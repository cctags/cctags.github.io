Git-P4 Submit Error

2015-09-30 Wednesday · By cctags · Posted in VCS · 0 Comments

提交代码时遇到了这样的错误： 

  $ git p4 submit
  Error: Cannot locate perforce checkout of //XXX/YYY/ZZZ/ in client view

一下子还真没想明白是哪里不对。

后来想明白了，需要重新登录一下 Perfoce： 

  $ p4 login
  ...

解决。
