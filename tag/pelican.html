Use Google Analytics With Pelican

Posted on 2020-04-13 星期一 in Tools • Tagged with Pelican

基于 Pelican 生成的静态页面，可以使用 Google Analytics 来统计访问数据。

以下是操作步骤，记录在这里备忘：

  1. 登录到 Google Analytics，创建账号；
  2. 添加需要统计的 URL；
  3. 获取 Tracking ID；
  4. 把 Tracking ID 添加到 publishconf.py 里： 

      GOOGLE_ANALYTICS = 'UA-XXXXXX-Y'

  5. 重新编译并发布即可。

参考：Google Analytics With Pelican

Pelican 编译问题

Posted on 2017-12-22 星期五 in Tools • Tagged with Pelican, Python

今天遇到了一个问题：重新编译 pelican 项目时，发现编译生成的 articles 数量要比正常的少很多，而且没有任何出错信息。

经过研究，找到了原因。前段时间更新了 Python 2.7 到最新版本，并且安装了第三方的库，当然包括了 pelican。但 markdown 并没有安装，这导致在编译时 markdown 文件时无法解析。

解决方案很简单，安装 markdown，然后重新编译即可。

pip install markdown

之后又重新看了一遍 pelican 安装说明，明确了 markdown 只属于 optional packages，而不是必选的依赖项。

Setup Comments with Disqus in Pelican

Posted on 2015-09-18 星期五 in Tools • Tagged with Pelican, Disqus

Disqus 是一个外挂的评论系统，对于像 GitHub Pages 这种静态网页来说，非常实用。

找到一个文档 http://querbalken.net/howto-setup-comments-with-disqus-in-pelican-en.html，把Disqus的使用步骤写得很清楚了。

以下是主要的配置说明，记录在这里，备忘。

注册 Disqus 账号：通过链接：http://<YOUR_DOMAINE>.disqus.com/admin/settings，进行设置。

配置 Pelican：在 pelicanconfig.py 文件里，添加如下配置：

DISQUS_SITENAME = "<YOUR_DOMAINE>"

Pelican 3.6 发布了。

Use txt2tags in Pelican

Posted on 2015-06-07 星期日 in Tools • Tagged with Pelican, txt2tags

1. txt2tags

Markdown 是 Pelican 默认支持的标记语言。其语法简洁，功能实用，多数情况下，可以满足实际的使用需求。

txt2tags 是一个类似的项目： 

  Txt2tags is a text formatting and conversion tool.

相比于 Markdown：

  • txt2tags 包括了以下这些有用的功能：
    • 可以指定生成目录，比如 %!options: -n --toc
    • Numbered Title
    • Table
    • Macros

  • 当然也有一些不足：
    • Verbatim line 没有语法高亮

更多功能，请参考用户手册。

总的来说，txt2tags 的这些功能，可以使得内容更加清晰，有条理。
